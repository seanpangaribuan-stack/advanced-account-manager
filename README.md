# advanced-account-manager

Welcome to the **Advanced Accounts and Group Policy Manager v2.0.0**! This is a professional system utility built in an executable format (`.exe`) to manage your Windows environment efficiently.

### 💎 Key Features:
* **Identity Modification**: Change your local account username instantly via core system tools.
* **Live Hardware Diagnostics**: Pulls live processor name and RAM telemetry straight from Windows Settings.
* **Storage Allocation**: Live tracking of your logical disk (C:) capacity.
* **Network Interrogation**: Displays your active virtual adapter configuration and IP address.
* **Update Manager Simulation**: Built-in visual check system to simulate cloud repository connections.

### ⚠️ Execution Requirement:
To prevent access violations and allow local SAM database modifications, you **must right-click** `AccountManager.exe` and select **"Run as administrator"**.
