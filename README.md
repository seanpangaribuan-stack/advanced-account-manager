# Windows Advanced Core Diagnostics, Repair and Security Suite v3.0.0

Welcome to the **Ultimate Global Release (v3.0.0)** of the Advanced Account and System Manager! This utility has been fully upgraded from a simple credential reassigner into a comprehensive **All-in-One Tuning & Security Suite** packaged in a secure executable (`.exe`) format.

### 🎨 Custom Design Branding
* **Official Logo**: Embedded with a custom-designed blue Windows window and an ash-gray gear element (32x32 pixels, true transparency).
* **Architecture**: Optimized strictly for 64-bit Windows environments.

### 💎 The 10 Core Operational Sectors:
1. **View Current User Profile Properties**: Live tracking of active session metadata, user domain, and profile registry paths.
2. **Modify Local Account Identity**: Instant script-driven local machine username modifier via core security descriptors.
3. **System Update Manager**: Simulated cloud-handshake synchronization interface for deployment verification.
4. **Hardware & CPU Specifications**: Pulls live processor name telemetry and physical RAM capacity straight from the registry layer.
5. **Storage & Disk Capacity Details**: Real-time evaluation of partition allocation tables on your logical drives.
6. **Network & IP Configuration Status**: Interrogates the active local IP stack and adapter properties.
7. **Advanced System Repair Engine**: Automated emergency protocol deploying official **SFC Scan** and **DISM Component Store Repair** to eliminate file corruption.
8. **Automated Storage Cleaning Utility**: Instantly purges buried system caches (`Temp` & `Prefetch`) and fires up configuration tools to prevent heavy disk bloating.
9. **Anti-Malware Protection Shield**: Direct integration with Windows Defender to execute live malware scans, grab definition updates, and engage anti-ransomware folder locking.
10. **Terminate Utility**: Safely close out directory operational services.

### ⚠️ Execution & Deployment Requirement:
Because this application modifies deep system files, security hashes, and active system shielding, it **requires full administrative privileges**. 
* **Compiled Version**: The `.exe` container is configured to automatically request access permissions.
* **Manual Script Version**: If executing via raw script, you **must right-click** `AccountManager.bat` and select **"Run as administrator"**.
